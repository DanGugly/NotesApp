# Simple Notes App in Kotlin
## This Application utilizes the following technologies:
- ### Kotlin
- ### Coroutines
- ### ViewModel
- ### RoomDB
- ### Jetpack Navigation
- ### DataStore
- ### Compose
- ### Dependancy Injection with Dagger

## Users are able to:
- ### Add Notes
- ### Edit Notes
- ### Delete Notes
- ### Sort Notes
